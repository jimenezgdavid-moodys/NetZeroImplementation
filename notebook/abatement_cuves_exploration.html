Why are there regressions in the abatement curves?

total_AUC = function(x,y){
  
  n = length(x)
  
  x2 = x[-1]
  x1 = x[-n]
  y2 = y[-1]
  y1 = y[-n]
  
  sum((y1+y2)*(x2-x1)/2)
  
}
